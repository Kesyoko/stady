**```docker compose exec web bundle install ```と``` docker compose exec web bundle exec rails g rspec:install ```の違い**<br>
大抵のGemはGemファイルに記載した後、bundle installを行えば良いがRspecやsorceryなどは初期設定のために↑を行う必要がある。<br>
初期設定を行わないGemはbundle installを行うだけで良い。<br>

``~password_confirmation: "password"``<br>
上記のコードを入力することによって確認のため再入力された文字とあっているか確認することができる。（今回の例ではパスワードが対象、メアドを確認したいときはバリデーションからいじる必要がある。）
