mv ~<br>
**mv 元ファイル名　変更後ファイル名名称**とすることですでに制作したファル名を変更することができる。<br>
```ex)mv app/views/admin/boards/_boards.html.erb app/views/admin/boards/_board.html.erb```<br>

エラーに対策に使えるコマンド<br>
```
##### ログ確認
→ tail -50 log/development.log
 
ルーティング確認
→ docker compose exec web rails  routes 
(コントローラー名で絞りたい時) docker compose exec web rails  routes | grep [コントローラー名]
 
Railsコンソール
→ docker compose exec web rails console 
　
ファイル全体を表示
→ cat app/controllers/posts_controller.rb
　
直前のコマンドを再実行
→ !!
```
##### データーベースエラー時<br>
```
マイグレーションは最新か・UPになっているか
docker compose exec web  rails db:migrate:status

テーブルは存在するか
docker compose exec web rails console
> ActiveRecord::Base.connection.tables (76678_Kesyoko_basic_rails_basic/db/schema.rbに書かれているBDが表示される)

# 3. データは入ってるか
rails console
> Post.count (Postクラスに何個データが入っているか)
> User.first
```
 
エラー解決方法<br>
エラーでたらエラーメッセ確認→docker compose exec web rails  routes でルーティング確認→tail -50 log/development.logでログを確認、修正
