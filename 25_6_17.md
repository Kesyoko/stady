**turbo_method**<rb>
リンクをクリックするとGETが送られるが、これを使用することによってGET以外のHTTPメソッドを使用できるようにするためのメソッド。<rb>
```
<%= link_to "投稿を削除", post_path(post), data: { turbo_method: "delete", turbo_confirm: "削除してよろしいですか？" } %>

```
と記載することによってGET以外を使用することができ、**turbo_confirm**の後ろにメッセージを入れることで消去ボタンを押した時にメッセージを表示することができる。<rb>
　<rb>
復習）HTTPメソッドは以下の通り。

| やりたいこと | HTTPメソッド | 例 |
| --- | --- | --- |
| ページを取得したい | GET | リンクを踏む |
| 新しいデータを登録したい | POST | フォーム新規登録 |
| 既存データを更新したい | PATCH/PUT | フォーム編集保存 |
| データを削除したい | DELETE | 削除ボタン |
